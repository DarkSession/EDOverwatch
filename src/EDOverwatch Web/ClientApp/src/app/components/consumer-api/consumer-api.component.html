Check out the following REST endpoints.
Please note that you need to add at the end of the endpoint if you are calling these endpoints in your browser. This is to bypass the site service worker in your browser.

Overview
GET /api/v1/overwatch/overview

List systems
GET /api/v1/overwatch/systems

Get system details by system address
GET /api/v1/overwatch/system/1694104537435

List titans
GET /api/v1/overwatch/titans

List weekly cycles
GET /api/v1/overwatch/thargoidcycles

List system states by weekly cycle (start date)
Invalid dates or dates in the future return the current cycle.
GET /api/v1/overwatch/systems/2023-01-05

Alert predictions for the next cycle
GET /api/v1/overwatch/alertpredictions
