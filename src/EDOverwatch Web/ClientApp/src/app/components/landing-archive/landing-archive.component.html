Overwatch Archival
All eight Titans have been defeated and there are currently no systems which are affected by any of the Thargoid war states.

With the war coming to an end, DCoH Overwatch has served its purpose and is now read-only.

What happens to Overwatch?
Many would agree that we probably have not seen the last of the Thargoids. However, we do not know if the Thargoid war states as we have seen them the last 2 years will remain in-game.
For now, Overwatch has been set to read-only but can be reactivated at any time should we see Titans and Thargoid war states again.
If the game is going to phase out the Thargoid war states, we will keep Overwatch as read-only and eventually shut it down within a 24 - 48 months timeframe.
The API will remain available in the read-only phase.

What happens with all the Thargoid war information stored in Overwatch?
All war related information will be exported and made available on Github.
This will likely take another 1-2 weeks from the time you are seeing this notice.
