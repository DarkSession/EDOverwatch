About
DCoH Overwatch/ED: Overwatch is a project developed and maintained by CMDR Dark Session in collaboration with the Defense Council of Humanity team.

API
All data available in Overwatch should be available via API as well.
API data can be used for free without the need of crediting DCoH Overwatch.
API data is provided "as-is" without any guarantee of accuracy.
Our only ask is that all API consumers keep the number of API requests reasonable.
